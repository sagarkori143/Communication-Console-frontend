Custom SMS Template Documentation

Introduction

Registering a Template on DLT Platform:
  1. Login to your DLT platform (e.g., Vodafone, Airtel, Jio).
  2. Navigate to the "Template Registration" section.
  3. Provide the template name, content, and category (Promotional/Transactional).
  4. Specify variable placeholders.
  5. Submit the template for approval.
  6. Once approved, use the template ID for sending SMS via your service provider.

Components

Main Features

The platform offers several features:

  • Create Template: Create new SMS templates.
  • Configure Template: Modify existing templates.
  • Select User : Select users for SMS targeting.
  • History: View SMS sending history.

Usage

Navigation

Use the menu to change the selectedMenu variable, controlling which feature is displayed.

Creating a Template

  1. Select "Create Template" from the menu.
  2. Fill in the form and submit to save the template.

Configuring a Template

  1. Select "Configure Template" from the menu.
  2. Choose a template and update as needed.

Viewing & Uploading User Data

Select "View User Data" to filter and target users, or "Upload User Data" to add new data for targeting and send Sms

Viewing History

View "History" to track sent messages, timestamps, and recipients.

Conclusion

This platform simplifies SMS notifications, allowing users to efficiently create, send, and track messages.
